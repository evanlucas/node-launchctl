# node-launchctl

[![Build Status](https://travis-ci.org/evanlucas/node-launchctl.png?branch=master)](https://travis-ci.org/evanlucas/node-launchctl)

Provides native bindings to launchctl commands

## Dependencies

- Tested on OS X 10.8+
- Requires Xcode 4.5+

## Install

    npm install launchctl

## Test

    grunt
